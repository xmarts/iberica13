Asiento destino

Al realizar asientos con las cuentas 60,61,62,63,64,65,67,68 estos se denominan Asientos por Naturaleza.

Y el uso de estas cuentas generan Asientos de Destino, donde se carga a las cuentas Elemento 2 o por Centro de Costos, Elemento 9, con abono a la cuenta 61 o 79 respectivamente.

Configuración

Contabilidad

Ingrese a Contabilidad/Configuración/Plan Contable:

  • Elija la cuenta contable y active la opción "Tiene cuenta destino"
  • Elija las cuentas contables para generar el asiento destino
